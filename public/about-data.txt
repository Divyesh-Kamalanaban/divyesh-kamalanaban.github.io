Project: SleeQC — Resource-Adaptive Post-Quantum Cryptography on ESP32
Developed a continually developing system demonstrating resource-adaptive cryptography on an ESP32-S3 microcontroller.
Implemented a dynamic Post-Quantum Cryptography (PQC) engine that balances high security with real-time stability.
Integrated a TinyML decision module using TensorFlow Lite to monitor system load and autonomously switch cryptographic parameters.
Deployed ML-DSA-44 and ML-DSA-87 (Dilithium variants) for dynamic security scaling.
Resolved critical stack overflow issues and optimized system memory.
Developed firmware inculcating custom quantized ML algorithms trained on real-word resource activity data.
Ensured system reliability through task-level profiling and adaptive switching.
Handled complex C/C++ linker issues within the ESP-IDF framework and coordinated multi-language integration.
Link: https://github.com/Divyesh-Kamalanaban/sleeqc

Project: Gridifix - ML-Powered Fault Detection and Localization Engine for Smart Grids
Designed a research pipeline for machine-learning-based grid fault detection and localization.
Affiliated with Divyesh's internship at NIT Trichy.
Generated over 21,000 simulated power grid scenarios including controlled faults and variable loads using pandapower.
Trained deep neural networks on 14,000 samples, achieving 95% accuracy in power flow estimation.
Implemented ensemble models reaching 92% fault detection and 100% localization accuracy.
Enhanced solver performance by integrating a DLPF-based optimization, reducing computation time by 70% compared to Newton–Raphson.
Link: https://github.com/Divyesh-Kamalanaban/gridifix

Project: Seaquenced - Identifying Taxonomy and Assessing Biodiversity from eDNA Datasets
Contributed to Smart India Hackathon 2025 under the bioinformatics category.
Built a classification pipeline to identify species from environmental DNA (eDNA) data using machine learning.
Expanded raw dataset from 5 FASTQ species files by applying mutation-based augmentation, yielding 10,000+ synthetic samples.
Quantized Nucleotide Transformer models from 32-bit to 4-bit precision, cutting GPU memory usage from 9.5GB to 2.5GB with minimal accuracy loss.
Supported multi-stage collaboration including preprocessing, model training, and integration.
Link: https://github.com/Divyesh-Kamalanaban/seaquenced
